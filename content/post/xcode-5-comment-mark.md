XCode 5 內建支援的標記有

```
// FIXME: I really need to fix this
// TODO: make some money
// MARK: mom, I am here !
// ???: what r u doing?
// !!!: this is magic code.
```

比對的條件是 `// <Keyword>:` 冒號是必要的
